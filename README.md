# Getting Started

Your portfolio website has been created with the following structure:

## Files Included:

- **index.html** - Main landing page with all sections (About, Projects, Contact)
- **styles.css** - Complete styling with responsive design
- **_config.yml** - GitHub Pages configuration

## Next Steps to Deploy:

1. **Initialize a Git Repository** (if not already done):
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio commit"
   ```

2. **Create a GitHub Repository**:
   - Go to github.com and create a new repository
   - Name it: `yourusername.github.io`
   - Push your local files to GitHub

3. **Update Your Information**:
   - Replace placeholder text in `index.html` with your actual information:
     - "Your Name" and "Your Professional Title"
     - About section content
     - Skills and technologies
     - Project details
     - Contact information and links

4. **GitHub Pages will automatically deploy** your site to: `https://yourusername.github.io`

## Customization Tips:

- **Colors**: Modify the color values in `styles.css` (look for hex codes like #667eea)
- **Add More Projects**: Duplicate the `project-card` div in the projects section
- **Add More Skills**: Add new `<li>` items in the skills list
- **Add Images**: Create an `images/` folder and reference images in the HTML
